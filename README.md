Project Description
This is an example of project based on Cypress. The project contains tests for web-services:

create user endpoint provided by https://reqres.in/
web-services for per entity provided by https://reqres.in/ Testing data is generated randomly using chance library  and existing test data.

And there is also one UI tests for web-site https://www.amazon.com/ Search by product name scenarios is implemented. It uses data from the fixture/TestData file.
